# 更新日志 (Changelog)

## v0.2.0 (2026-04-24) - 多语言版 + 腾讯地图 API + 数据扩充

### 🎉 首次发布

这是广州旅游 Skill 的首个正式版本，包含完整的多语言支持、腾讯地图 API 集成和丰富的旅游数据。

---

### ✨ 新增功能

#### 🌍 多语言支持
- ✅ 支持联合国 6 大官方语言：
  - 🇨🇳 中文 (zh)
  - 🇺🇸 英文 (en)
  - 🇫🇷 法文 (fr)
  - 🇷🇺 俄文 (ru)
  - 🇪🇸 西班牙文 (es)
  - 🇸🇦 阿拉伯文 (ar)
- ✅ 所有 API 接口支持多语言切换
- ✅ 语言回退机制（不支持的语言自动回退到中文）
- ✅ 多语言数据字段（name_en, description_en 等）

#### 🗺️ 腾讯地图 API 集成
- ✅ 地点搜索（酒店、餐饮、学校、医院等）
- ✅ 逆地址解析（坐标转地址）
- ✅ 地址解析（地址转坐标）
- ✅ 距离计算
- ✅ POI 数据格式化
- ✅ API Key 环境变量管理（安全配置）

#### 🛠️ MCP Server 工具（12 个）

**本地数据工具（6 个）**
1. `get_attractions` - 获取景点信息
2. `get_food_guide` - 获取美食推荐
3. `get_history` - 获取历史故事
4. `get_culture` - 获取人文风情
5. `get_stories` - 获取城市故事
6. `get_supported_languages` - 获取支持的语言列表

**地图工具（6 个）**
7. `get_hotels` - 获取酒店推荐
8. `get_transport` - 获取交通信息
9. `get_guangzhou_map_overview` - 获取广州地图概览
10. `tencent_map_search` - 腾讯地图地点搜索
11. `tencent_map_reverse_geocoder` - 逆地址解析
12. `tencent_map_geocoder` - 地址解析
